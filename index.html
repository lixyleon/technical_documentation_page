Introduccion

MySQL es un sistema de administración de bases de datos (Database Management System, DBMS) para bases de datos relacionales open source. Se sustenta en SQL: Structure Query Language (Lenguaje de Consulta Estructurado).

MySQL utiliza múltiples tablas para almacenar y organizar la información. Fue escrito en C y C++ y destaca por su gran adaptación a diferentes entornos de desarrollo, permitiendo la interacción con los lenguajes de programación de más uso como PHP, Perl y Java; así como, su integración en distintos sistemas operativos.
